# Introducción — Panel Administrador MiTienda.pe

## ¿Qué es MiTienda.pe?

MiTienda.pe es una plataforma para crear y gestionar tu tienda online. Desde el Panel Administrador puedes controlar todo: tus productos, pedidos, pagos, envíos, marketing y la apariencia de tu tienda.

## Acceso al Panel Administrador

1. Ingresa a [admin.mitienda.pe](https://admin.mitienda.pe)
2. Escribe tu email y contraseña
3. Si tienes acceso a múltiples tiendas, selecciona la tienda con la que deseas trabajar

## Estructura del Panel

El panel está organizado en secciones accesibles desde el menú lateral:

| Sección | Qué encontrarás |
|---|---|
| **Dashboard** | Resumen de ventas y actividad reciente |
| **Clientes** | Base de compradores |
| **Ventas** | Pedidos, reviews, reclamos, carritos abandonados |
| **Reportes** | Exportaciones e informes detallados |
| **Catálogo** | Productos, categorías, marcas, variantes |
| **Marketing** | Promociones, cupones, combos, referidos |
| **Contenido** | Páginas, blog, banners, galería |
| **Apariencia** | Diseño, colores, tipografía, menú |
| **Formas de Pago** | Pasarelas y métodos de pago |
| **Envíos** | Zonas, tarifas y couriers |
| **Facturación** | Comprobantes electrónicos (SUNAT) |
| **Tu Tienda** | Configuración, dominio, SEO, usuarios |
| **Integraciones** | Email marketing, CRM, webhooks |

## Primeros pasos recomendados

Para poner tu tienda en línea sigue este orden:

1. **Información de la tienda** — nombre, logo, datos de contacto
2. **Formas de pago** — activa al menos un método de pago
3. **Envíos** — configura zonas y tarifas de despacho
4. **Catálogo** — crea tus categorías y carga tus productos
5. **Apariencia** — personaliza colores, tipografía y menú
6. **Facturación** — configura si emitirás comprobantes electrónicos

## Módulos disponibles según tu plan

Algunos módulos están disponibles solo en ciertos planes de suscripción. Si ves una sección bloqueada, puedes actualizar tu plan desde **Tu Tienda → Plan y Suscripción**.
