# Sucursales y Direcciones

**Ruta:** Panel Administrador → Tu Tienda → Sucursales

## ¿Para qué sirven las Sucursales?

Las sucursales tienen dos usos principales:
1. **Punto de recojo** — aparecen como opción en el checkout para que el comprador retire su pedido
2. **Dirección de remitente** — se usa en las etiquetas de envío como dirección de origen

## Crear una sucursal

| Campo | Descripción |
|---|---|
| Nombre | Nombre de la sucursal (ej: "Tienda Miraflores") |
| Dirección | Dirección completa |
| Ubigeo | Departamento / Provincia / Distrito |
| Teléfono | Teléfono de la sucursal |
| Horario de atención | Días y horas de apertura |
| Coordenadas | Latitud y longitud (para mapa) |
| Es punto de recojo | Si aparece como opción de pickup en el checkout |
| Es dirección de remitente | Si se usa como origen en las etiquetas de envío |

## Preguntas frecuentes

**¿Puedo tener varias sucursales activas como punto de recojo?**
Sí. El comprador podrá elegir en cuál de ellas desea recoger su pedido.

**¿Necesito tener sucursales para usar el módulo de Envíos?**
Para **envío a domicilio** no es necesario. Para **recojo en tienda**, sí debes tener al menos una sucursal marcada como punto de recojo.
