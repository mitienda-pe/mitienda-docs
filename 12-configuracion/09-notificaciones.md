# Notificaciones

**Ruta:** Panel Administrador → Notificaciones

## ¿Qué puedes configurar?

Define cómo y dónde recibes las alertas de tu tienda.

## Campos disponibles

| Campo | Descripción |
|---|---|
| Email de notificaciones | Email donde recibirás alertas de nuevos pedidos |
| Notificaciones push (OneSignal) | App ID y API key de OneSignal para push en el dispositivo |
| Google Analytics | ID de seguimiento de GA4 (ej: G-XXXXXXXX) |
| Google Tag Manager | ID del contenedor GTM (ej: GTM-XXXXXXX) |

## Notificaciones de nuevos pedidos

Cada vez que entra un pedido a tu tienda, el sistema envía un email al correo configurado. Puedes usar un email diferente al de tu cuenta de acceso.

## Notificaciones push

Si usas la App Móvil de MiTienda, puedes configurar OneSignal para recibir notificaciones en tu celular cuando entra un pedido.

## Google Analytics y GTM

- **Google Analytics (GA4):** Integración directa para rastrear tráfico y comportamiento en tu tienda
- **Google Tag Manager:** Si prefieres gestionar todos tus scripts (GA4, Pixel, etc.) desde GTM, usa este campo

> Si usas GTM, no es necesario configurar GA4 aquí por separado: instala GA4 dentro de GTM.

## Preguntas frecuentes

**¿Puedo recibir notificaciones en varios emails?**
Por ahora solo se puede configurar un email de notificaciones. Si necesitas notificar a más personas, usa una lista de distribución de email.

**¿Las notificaciones push son gratuitas?**
OneSignal tiene un plan gratuito generoso. Consulta sus límites en onesignal.com.
