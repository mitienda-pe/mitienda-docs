# TikTok

**Ruta:** Panel Administrador → Integraciones → Proveedores → TikTok Pixel

::: info Acceso desde Integraciones
Esta configuración se accede desde el módulo de **Integraciones → Proveedores**. Haz clic en la tarjeta de TikTok para llegar aquí.
:::

## ¿Para qué sirve?

Conecta tu tienda con TikTok para rastrear conversiones de tus anuncios en esta plataforma.

## Campos disponibles

| Campo | Descripción |
|---|---|
| TikTok Pixel ID | ID del píxel de TikTok para seguimiento de conversiones |

## Cómo obtener el TikTok Pixel ID

1. Ingresa a **TikTok Ads Manager**
2. Ve a **Herramientas → Eventos**
3. Crea o selecciona tu píxel web
4. Copia el Pixel ID y pégalo aquí

## Preguntas frecuentes

**¿Qué eventos trackea el píxel de TikTok?**
MiTienda envía los eventos principales de e-commerce: visita a la tienda, vista de producto, agregar al carrito y compra completada.
