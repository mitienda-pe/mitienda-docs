# Blog

**Ruta:** Panel Administrador → Contenido → Blog

## ¿Qué es el Blog?

El blog te permite publicar artículos, noticias y contenido de valor para tus compradores. Es útil para SEO y para mantener comunicación con tu audiencia.

## Crear un post

Ve a **Contenido → Blog → + Nueva Entrada**.

| Campo | Descripción |
|---|---|
| Título | Título del artículo |
| Slug | URL del post |
| Contenido | Cuerpo del artículo (mismos 3 editores que Páginas) |
| Extracto | Resumen corto que aparece en el listado |
| Imagen destacada | Imagen principal del post |
| Categoría | Categoría del blog |
| Autor | Nombre del autor |
| Publicado | Si el post es visible |
| Meta título / descripción | Para SEO |
| Fecha de publicación | Puedes programar la fecha |

## Categorías del Blog

Puedes crear categorías para organizar tus posts. Ve a **Contenido → Blog → Categorías**.

Cada categoría tiene nombre, slug y puede tener descripción.

## Editores

Los mismos tres modos disponibles en Páginas: Visual (WYSIWYG), Código (HTML) y Visual Builder.

## Preguntas frecuentes

**¿El blog tiene página de inicio?**
Sí. Tu tienda tiene automáticamente una página de blog en `/blog` donde aparecen todos los posts publicados.

**¿Puedo programar la publicación de un post?**
Sí. Establece una fecha de publicación futura y el post no se mostrará hasta esa fecha.

**¿Las categorías del blog son las mismas que las de productos?**
No. Son categorías independientes, solo para el blog.
