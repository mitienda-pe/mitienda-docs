# Carrusel de Portada

**Ruta:** Panel Administrador → Contenido → Carrusel

## ¿Qué es el Carrusel?

El carrusel es el slider de imágenes que aparece en la página principal de tu tienda. Puedes tener hasta **6 slides** con imágenes y enlaces.

## Crear un slide

| Campo | Descripción |
|---|---|
| Texto alternativo | Texto alt de la imagen (para accesibilidad y SEO) |
| Imagen desktop | Imagen para pantallas grandes |
| Imagen mobile | Imagen para celulares (proporción diferente) |
| Enlace | URL a la que redirige al hacer clic |
| Orden | Posición del slide en el carrusel |

**Formatos de imagen:** JPG, PNG, WebP.

Se recomienda usar imágenes distintas para desktop y mobile para asegurar que el banner se vea bien en todos los dispositivos.

## Reordenar slides

Usa los **botones de flechas** (subir/bajar) para cambiar el orden en que aparecen los slides en el carrusel.

## Preguntas frecuentes

**¿Cuántos slides puedo tener?**
Máximo 6 imágenes. Se recomienda entre 3 y 5 para que el carrusel no sea demasiado largo.

**¿Qué tamaño deben tener las imágenes del carrusel?**
La proporción recomendada es **16:9** para desktop. El ratio es configurable desde la interfaz.

- Mobile: 768 x 500 px

Asegúrate de que el texto o elemento central de la imagen quede centrado para que se vea bien en ambos tamaños.
