# Carrusel / Banners

**Ruta:** Panel Administrador → Contenido → Carrusel

## ¿Qué es el Carrusel?

El carrusel es el slider de imágenes que aparece en la página principal de tu tienda. Puedes tener múltiples slides con imágenes y enlaces.

## Crear un slide

| Campo | Descripción |
|---|---|
| Título | Texto del slide (opcional) |
| Imagen desktop | Imagen para pantallas grandes |
| Imagen mobile | Imagen para celulares (proporción diferente) |
| Enlace | URL a la que redirige al hacer clic |
| Orden | Posición del slide en el carrusel |
| Activo | Si el slide se muestra o no |

**Formatos de imagen:** JPG, PNG, WebP.

Se recomienda usar imágenes distintas para desktop y mobile para asegurar que el banner se vea bien en todos los dispositivos.

## Reordenar slides

Arrastra los slides para cambiar el orden en que aparecen en el carrusel.

## Preguntas frecuentes

**¿Cuántos slides puedo tener?**
No hay un límite fijo, pero se recomienda entre 3 y 5 para que el carrusel no sea demasiado largo.

**¿Qué tamaño deben tener las imágenes del carrusel?**
- Desktop: 1920 x 600 px (relación 16:5 aproximada)
- Mobile: 768 x 500 px

Asegúrate de que el texto o elemento central de la imagen quede centrado para que se vea bien en ambos tamaños.
