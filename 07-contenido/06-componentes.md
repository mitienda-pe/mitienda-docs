# Componentes Reutilizables

**Ruta:** Panel Administrador → Contenido → Componentes

## ¿Qué son los Componentes?

Los componentes son bloques de contenido HTML que puedes crear una vez y reutilizar en varias páginas o en el blog. Por ejemplo: un bloque de "Garantía de devolución", un banner de WhatsApp o una sección de certificaciones.

## Crear un componente

| Campo | Descripción |
|---|---|
| Nombre | Nombre interno del componente |
| Contenido | HTML del bloque (con editor Visual o Código) |
| Activo | Si está disponible para insertar |

## Usar un componente en una página

Desde el editor de Páginas o Blog, puedes insertar un componente guardado en el lugar donde lo necesites.

## Preguntas frecuentes

**¿Si edito un componente, se actualiza en todas las páginas donde está insertado?**
Sí. Al editar el componente, el cambio se refleja en todos los lugares donde está siendo usado.

**¿Puedo usar componentes en el constructor visual (MTBuilder)?**
Sí, los componentes guardados están disponibles como bloques dentro del constructor visual.
