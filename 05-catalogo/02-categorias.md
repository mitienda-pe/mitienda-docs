# Categorías

**Ruta:** Panel Administrador → Catálogo → Categorías

## ¿Qué son las Categorías?

Las categorías organizan tu catálogo en grupos lógicos para que los compradores encuentren fácilmente lo que buscan. Puedes crear una jerarquía con categorías y subcategorías.

## Crear una categoría

Haz clic en **Nueva Categoría** y completa:

| Campo | Descripción |
|---|---|
| Nombre | Nombre visible en la tienda (ej: "Ropa de mujer") |
| Descripción | Texto descriptivo de la categoría (opcional) |
| Slug | URL amigable (ej: `ropa-de-mujer`), se genera automáticamente |
| Categoría padre | Si es una subcategoría, selecciona la categoría padre |
| Orden | Posición en el listado |
| Estado | Activa o inactiva |

### Imágenes de la categoría

Puedes subir tres tipos de imagen:
- **Imagen cuadrada** — para listados y miniaturas
- **Imagen cover** — para banners de portada de la categoría
- **Imagen Open Graph** — para compartir en redes sociales

## Subcategorías

Para crear una subcategoría, simplemente selecciona una **categoría padre** al crearla. No hay límite de niveles, aunque se recomienda no más de 3 niveles de profundidad para una navegación cómoda.

## Asignar productos a una categoría

Desde el detalle de la categoría puedes vincular o desvincular productos de forma masiva. También puedes asignar categorías directamente desde el formulario de cada producto.

## Orden de las categorías

Ajusta el orden en que aparecen las categorías en tu tienda editando el campo **Orden** de cada una. Menor número = aparece primero.

## Preguntas frecuentes

**¿Un producto puede estar en varias categorías?**
Sí. Un producto puede asignarse a múltiples categorías al mismo tiempo.

**¿Puedo crear una categoría vacía temporalmente?**
Sí. Puedes crear la categoría y agregarle productos después.

**¿Qué pasa si desactivo una categoría?**
Los productos dentro de ella siguen existiendo, pero la categoría deja de mostrarse en la navegación del storefront.

**¿Cómo agrego la categoría al menú de navegación?**
Desde **Apariencia → Menú** puedes agregar un enlace a cualquier categoría. Ver: [Menú de Navegación](../08-apariencia/05-menu.md).
