# Listas de Productos

**Ruta:** Panel Administrador → Catálogo → Listas

## ¿Qué son las Listas?

Las listas son colecciones curadas de productos que puedes mostrar en la página principal de tu tienda. Ejemplos típicos: "Novedades", "Más vendidos", "Ofertas de la semana".

## Crear una lista

| Campo | Descripción |
|---|---|
| Nombre | Título de la colección (ej: "Más vendidos") |
| Descripción | Texto descriptivo opcional |
| Orden | Posición en la homepage |
| Activa | Si se muestra o no en el storefront |
| Productos | Productos que forman parte de la lista |

## Agregar productos a una lista

Desde el detalle de la lista puedes buscar y vincular los productos que deseas incluir. También puedes quitarlos cuando ya no los necesites.

## Preguntas frecuentes

**¿Las listas se actualizan automáticamente?**
No. Son colecciones manuales. Tú decides qué productos incluir y cuándo actualizarlas.

**¿Cuántas listas puedo tener en la homepage?**
No hay un límite fijo, pero se recomienda no mostrar más de 3-4 para mantener una página limpia.

**¿Puedo enlazar una lista desde el menú?**
Sí, desde **Apariencia → Menú** puedes agregar un enlace a cualquier lista activa.
