# Etiquetas

**Ruta:** Panel Administrador → Catálogo → Etiquetas

## ¿Qué son las Etiquetas?

Las etiquetas (tags) son palabras clave que puedes asignar a tus productos para facilitar la búsqueda y agrupación. A diferencia de las categorías, son más flexibles y un producto puede tener muchas etiquetas.

## Crear una etiqueta

| Campo | Descripción |
|---|---|
| Nombre | Texto de la etiqueta (ej: "oferta", "nuevo", "verano 2025") |
| Slug | URL generada automáticamente |

## Asignar etiquetas a productos

Puedes asignar etiquetas desde el formulario de cada producto (campo **Etiquetas**) o hacer asignación masiva desde este módulo.

## Preguntas frecuentes

**¿Cuántas etiquetas puede tener un producto?**
No hay límite. Asigna todas las que sean relevantes.

**¿Las etiquetas son visibles en la tienda?**
Depende de cómo esté configurada tu tienda. Pueden usarse para filtros de búsqueda internos o mostrarse como tags en la página del producto.
