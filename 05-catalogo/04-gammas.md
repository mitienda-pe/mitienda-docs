# Gamas / Líneas

**Ruta:** Panel Administrador → Catálogo → Gamas

## ¿Qué son las Gamas?

Las gamas (o líneas de producto) son un nivel de clasificación adicional dentro de una marca. Por ejemplo: la marca "Nike" puede tener las gamas "Air Max", "React" y "Free Run".

## Crear una gama

| Campo | Descripción |
|---|---|
| Nombre | Nombre de la gama o línea |
| Descripción | Texto descriptivo (opcional) |
| Imagen | Imagen representativa |
| Marca relacionada | La marca a la que pertenece esta gama |
| Estado | Activa o inactiva |

## Vincular productos

Desde el detalle de la gama puedes gestionar los productos que pertenecen a ella. También puedes asignar la gama directamente desde el formulario de cada producto.

## Preguntas frecuentes

**¿Es obligatorio usar gamas?**
No. Es un nivel de clasificación opcional. Úsalas si tu catálogo tiene suficiente variedad dentro de cada marca para justificar sublíneas.

**¿Una gama siempre pertenece a una marca?**
Sí, las gamas se asocian a una marca. Si no usas marcas, tampoco necesitas gamas.
