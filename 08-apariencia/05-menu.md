# Menú de Navegación

**Ruta:** Panel Administrador → Apariencia → Menú

## ¿Qué es el Menú?

El menú de navegación es la barra principal de tu tienda que permite a los compradores moverse entre las secciones. Puedes construirlo libremente con hasta 3 niveles de jerarquía.

## Agregar un item al menú

Haz clic en **+ Agregar enlace** y configura:

| Campo | Descripción |
|---|---|
| Etiqueta | Texto visible en el menú (ej: "Ofertas") |
| Tipo de enlace | Qué tipo de página enlaza |
| URL destino | Se completa automáticamente según el tipo |
| Abrir en nueva pestaña | Si el enlace abre en una nueva ventana |

## Tipos de enlace disponibles

| Tipo | Enlaza a |
|---|---|
| Categoría | Página de una categoría de productos |
| Subcategoría | Página de una subcategoría |
| Producto | Página de un producto específico |
| Marca | Página de una marca |
| Gamma | Página de una gamma (sub-marca) |
| Lista | Una lista de productos curada |
| Página | Una página personalizada (ej: "Quiénes somos") |
| Categoría de blog | Listado de posts de una categoría del blog |
| Post de blog | Un artículo específico del blog |
| URL externa | Cualquier URL personalizada |

## Crear submenús (jerarquía)

Para crear un ítem hijo (submenú desplegable):

1. Agrega el ítem
2. En el campo **Elemento padre**, selecciona el ítem bajo el cual quieres anidarlo

Se soportan hasta **3 niveles**: padre → hijo → nieto.

## Reordenar el menú

Arrastra los ítems para cambiar su orden. Los cambios se guardan al hacer clic en **Guardar menú**.

## Edición inline

Puedes hacer doble clic sobre el label de cualquier ítem para editarlo directamente sin abrir un formulario.

## Preguntas frecuentes

**¿Cuántos ítems puedo tener en el menú?**
No hay un límite técnico, pero se recomienda no más de 7-8 ítems en el primer nivel para que el menú sea fácil de usar.

**¿Puedo tener un menú diferente para mobile?**
Por ahora hay un único menú que se adapta a todos los dispositivos (responsive).

**¿Puedo agregar un ícono a los ítems del menú?**
No directamente desde el panel. Algunos temas permiten íconos, pero se configura a nivel de código.

**¿Los cambios en el menú se reflejan de inmediato?**
Sí, al guardar los cambios se aplican en el storefront inmediatamente.
