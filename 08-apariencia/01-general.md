# Configuración de la Tienda

**Ruta:** Panel Administrador → Apariencia → Configuración

## ¿Qué puedes configurar aquí?

Esta sección es el punto de partida para la identidad visual de tu tienda.

## Logo

Sube el logo principal de tu tienda. Este aparece en el header del storefront.

**Formatos aceptados:** PNG, SVG, JPG, WebP.

> Se recomienda usar PNG o SVG con fondo transparente para que el logo se vea bien sobre cualquier color de header.

## Favicon

El favicon es el pequeño ícono que aparece en la pestaña del navegador.

**Formatos aceptados:** PNG cuadrado de 512×512 px o SVG.

## Catálogo

Configura la visualización del catálogo de productos:

- **Columnas del catálogo** — selecciona cuántas columnas mostrar en el listado de productos (3, 4 o 6 columnas)

## Preguntas frecuentes

**¿Qué tamaño debe tener el logo?**
Se recomienda una imagen horizontal de al menos 200×60 px. El sistema la adapta automáticamente.

**¿Puedo tener logos diferentes para la versión desktop y mobile?**
Por ahora se usa el mismo logo en ambas versiones.
