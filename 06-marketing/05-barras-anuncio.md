# Barras de Anuncio

**Ruta:** Panel Administrador → Marketing → Barras de Anuncio

## ¿Qué son las Barras de Anuncio?

Son banners en la parte superior de tu tienda (antes del header) para comunicar mensajes importantes: promociones, envío gratis, fechas especiales, etc.

## Crear una barra de anuncio

| Campo | Descripción |
|---|---|
| Nombre interno | Identificador interno de la barra (no visible para el comprador) |
| Texto del anuncio | El mensaje que verán los visitantes |
| Enlace | URL a la que redirige al hacer clic (opcional) |
| Color de fondo | Color de la barra |
| Color del texto | Color de las letras |
| Posición | Superior (antes del header) o Inferior (al final de la página) |
| Agregar botón de acción | Toggle para agregar un botón con texto y enlace personalizado |
| Permitir cerrar la barra | Toggle para que el visitante pueda cerrar la barra |
| Fecha inicio / fin | Periodo de vigencia (opcional) |
| Estado | Activa o inactiva |

El formulario incluye una **vista previa en tiempo real** para que puedas ver cómo quedará la barra antes de guardar.

## Listado de barras

La tabla del listado muestra las columnas: Nombre, Texto, Posición, Vista previa, Estado y Programación.

## Preguntas frecuentes

**¿Puedo tener varias barras activas al mismo tiempo?**
Sí, pero solo una se muestra a la vez. El sistema muestra la de mayor prioridad o la más reciente.

**¿Las barras de anuncio afectan el SEO?**
El texto de la barra puede ser indexado por los buscadores, así que escríbelo pensando también en las palabras clave relevantes para tu tienda.
